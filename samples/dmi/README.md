# Sample DMI Files

This folder keeps example `.dmi` files in the repository for manual exploration, debugging, and ad-hoc validation.

Important:

- the application does not auto-load these files at startup
- the startup workflow still begins with an empty workspace
- these assets are optional examples, not a runtime dependency

The set intentionally includes both normal and edge-case inputs, for example:

- `default.dmi`, `color32x.dmi`, `color64x.dmi`
- body/clothing examples such as `testBodyHuman32x.dmi` and `testClothingDefaultCoat.dmi`
- edge cases such as `Empty.dmi` and `testBodyHumanBad.dmi`

Open them manually from the UI when you want a known sample input.
