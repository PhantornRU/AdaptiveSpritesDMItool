# Local Task State

Эта папка предназначена для локального рабочего состояния агента и намеренно частично исключена из git.

## Что можно создавать здесь
- `PLAN.md` для decision-complete плана по текущей задаче
- `TODO.md` для короткого списка следующих шагов
- `DECISIONS.md` для принятых решений и зафиксированных компромиссов
- `EVIDENCE.md` для краткой сводки проверок и ссылок на сырые логи
- `logs/` для длинного вывода команд, тестов и экспериментов

## Правила
- Держать эти файлы короткими и привязанными к текущему scope.
- Не использовать их как постоянную память проекта.
- Все долгоживущие правила и выводы переносить в стабильные документы в `AGENTS/`.
- Не коммитить локальные task-state файлы, если задача специально не требует обратного.
