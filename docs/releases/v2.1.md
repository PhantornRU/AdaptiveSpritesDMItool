# Adaptive Sprites DMI Tool v2.1

This file contains ready-to-paste release text in Russian and English.

## Русская версия

Adaptive Sprites DMI Tool v2.1 - релиз Windows x64 для редактирования, предпросмотра и применения `.dmi` pixel-mapping конфигов.

### Что вошло в v2.1

- Добавлены русские и английские ресурсы интерфейса, а также переключение языка в настройках.
- Добавлены настройки оболочки: тема, язык, viewport редактора, поведение панелей workspace, скрытие неактивных Source-полотен и вписывание нескольких direction-полотен в область просмотра.
- Обновлен редактор направлений: режимы `Single`, `Parallel`, `All`, большие canvas layout для scope-режимов и selector отображаемых направлений.
- Улучшен batch workspace: локализованный интерфейс, выбор папок и файлов, фильтрация, статусы, run log, исключение output-папки из input-сканирования и предпросмотр `One DIR` / `All DIR`.
- Исправлены сценарии `Fill` и `Move`, включая сохранение mapping при переносе и корректное поведение зеркалирования и параллельных направлений.
- Улучшены rendering pipeline и производительность редактора: кэширование, batching частых обновлений и более стабильное поведение при рисовании и zoom.
- Исправлено корректное закрытие окна и сохранение состояния workspace при выходе.
- Добавлена локальная release-сборка ZIP и SHA256 для ручной публикации.
- Обновлена конфигурация VS Code debug: новый запуск использует C# Dev Kit `dotnet` debug type, а старый `coreclr` оставлен как legacy fallback.

## English Version

Adaptive Sprites DMI Tool v2.1 is a Windows x64 release for editing, previewing, and applying `.dmi` pixel-mapping configs.

### What's New In v2.1

- Added Russian and English UI resources, plus a language setting.
- Added shell/settings controls for theme, language, editor viewport, workspace panel behavior, hiding inactive Source canvases, and fitting multiple direction canvases into the viewport.
- Updated the direction editor with `Single`, `Parallel`, and `All` scopes, larger canvas layouts for scoped editing, and a direction display selector.
- Improved the batch workspace with localized UI, folder/file selection, filtering, status display, run log, output-folder exclusion from input scanning, and `One DIR` / `All DIR` preview.
- Fixed `Fill` and `Move` scenarios, including mapping preservation during moves and correct mirrored/parallel direction behavior.
- Improved editor rendering and performance through caching, batched frequent updates, and more stable drawing/zoom behavior.
- Fixed window shutdown and workspace state saving on close.
- Added local release ZIP and SHA256 generation for manual publication.
- Updated VS Code debug configuration: the default launch now uses the C# Dev Kit `dotnet` debug type, while the old `coreclr` launch remains available as a legacy fallback.

### Compatibility

- Windows x64
- self-contained .NET 8 WPF
- JSON config schema `version: 1`
- CSV remains an import-only compatibility format
