# Adaptive Sprites DMI Tool v2.2

This file contains ready-to-paste release text in Russian and English.

## Русская версия

Adaptive Sprites DMI Tool v2.2 - релиз Windows x64 для редактирования, предпросмотра и применения `.dmi` pixel-mapping конфигов.

### Что вошло в v2.2

- Добавлены импортированные DMI state-слои для Source и Editable поверхностей.
- Для импортированных state-слоев добавлены placement mode `Background` / `Overlay`, явный порядок слоев и индивидуальная opacity.
- Порядок, placement, opacity и назначения импортированных state-слоев сохраняются в workspace settings и восстанавливаются при запуске.
- Улучшена загрузка и сортировка states, чтобы выбранные и восстановленные state-наборы были стабильнее.
- Обновлен sample DMI набор для проверки state-loading сценариев.
- Сохранены изменения v2.1: русские/английские ресурсы UI, настройка языка, shell/settings controls, direction scopes `Single` / `Parallel` / `All`, улучшенный batch workspace, исправления `Fill` / `Move`, mirrored/parallel directions, rendering/performance и корректное закрытие приложения.
- VS Code debug использует C# Dev Kit `dotnet` debug type; старый `coreclr` fallback не требуется.
- Release workflow теперь создает основной ZIP с приложением и отдельный samples ZIP с полной папкой `samples/`.

### Совместимость

- Windows x64
- self-contained .NET 8 WPF
- JSON config schema `version: 1`
- CSV остается import-only форматом совместимости

## English Version

Adaptive Sprites DMI Tool v2.2 is a Windows x64 release for editing, previewing, and applying `.dmi` pixel-mapping configs.

### What's New In v2.2

- Added imported DMI state layers for Source and Editable surfaces.
- Added `Background` / `Overlay` placement mode, explicit layer order, and per-layer opacity for imported state layers.
- Imported state layer order, placement, opacity, and surface assignment are saved in workspace settings and restored on startup.
- Improved state loading and state ordering so selected/restored state sets behave more predictably.
- Updated the sample DMI set for state-loading validation.
- Includes the v2.1 work: Russian/English UI resources, language setting, shell/settings controls, `Single` / `Parallel` / `All` direction scopes, improved batch workspace, `Fill` / `Move` fixes, mirrored/parallel direction fixes, rendering/performance improvements, and clean shutdown fixes.
- VS Code debug uses the C# Dev Kit `dotnet` debug type; the old `coreclr` fallback is not required.
- The release workflow now creates the main application ZIP and a separate samples ZIP containing the full `samples/` folder.

### Compatibility

- Windows x64
- self-contained .NET 8 WPF
- JSON config schema `version: 1`
- CSV remains an import-only compatibility format
