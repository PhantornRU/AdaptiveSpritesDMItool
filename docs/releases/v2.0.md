# Version 2.0

Adaptive Sprites DMI Tool v2.0 - релиз Windows x64 приложения для редактирования и применения `.dmi` pixel-mapping конфигов.

## Кратко

v2.0 поставляется как WPF-приложение с JSON config format, batch processing и поддержкой `4-dir` / `8-dir` sprites.

Релиз предназначен для Windows x64 и поставляется как self-contained ZIP с опубликованным WPF-приложением.

## Release Asset

Основной asset:

- `AdaptiveSpritesDMItool-v2.0-win-x64.zip`

После распаковки запускайте:

- `AdaptiveSpritesDmiTool.Presentation.Wpf.exe`

SHA256-файл можно приложить рядом с ZIP для проверки целостности скачанного архива.

## Что нового

- Новый WPF MVVM shell.
- Старт с пустой рабочей областью.
- Ручное открытие `.dmi`.
- Поддержка base, landmark и overlay state sources.
- Pixel-mapping editor для `4-dir` и `8-dir` sprites.
- Инструменты редактирования: `Paint`, `Fill`, `Move`, `Erase`, undo, area undo и selection.
- Preview modes: base, landmark, overlay, composite, grid, text-grid.
- JSON config schema `version: 1` как основной формат.
- CSV import path для старых конфигов.
- Explicit validation resolution и direction compatibility.
- Deterministic batch processing с per-file statuses.
- Batch overwrite policies: `SkipExisting`, `OverwriteExisting`, `FailIfExists`.
- Workspace/settings persistence между запусками.

## Совместимость

- Target platform: Windows x64
- Runtime: self-contained .NET 8 WPF
- Primary config format: JSON
- JSON schema version: `1`
- CSV: import-only compatibility
